# AI Call Center Assistant

A multi-agent system for analyzing call center recordings and transcripts.

## Features (Coming Soon)
- Audio transcription (Whisper)
- Call summarization (GPT-4)
- Quality scoring (Multi-LLM)
- Abuse detection
- Workflow visualization

## Current Phase: 0 - Deployment Verification

This project follows a "Deploy First, Build Incrementally" approach where each phase ends with a working deployment on Hugging Face Spaces.

## Tech Stack
- **Orchestration**: LangGraph
- **LLM Framework**: LangChain
- **Observability**: LangSmith
- **Model Control**: LiteLLM
- **UI**: Streamlit
- **Deployment**: Docker
